# Calender for Students
**学生の為の機能が揃うカレンダー**を作ってます（現在開発中）<br />
Webアプリ（的なやつ）なのでアプリのインストールが不要です<br />
また、それぞれの端末から同じカレンダーを管理することも可能になる予定です
